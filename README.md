# Personal-Portfolio
Third-year Aerospace Engineering student at Toronto Metropolitan University (spacecraft stream), focused on mechanical subsystem design, avionics, and aircraft performance. This repo collects technical reports, design work, and code from coursework and design teams.
---

## Projects

### ✈️ RC3 2026 — Performance Sizing & Airfoil Selection
*Toronto Metropolitan Aerospace Design (TMAD), Delivery Stream — Jan 2026–Present*

Built a MATLAB performance-sizing model to derive takeoff speed, an 8 m/s stall speed target, 
power requirements, and wing loading. Evaluated 3 candidate airfoils in XFLR5, improving lift 
coefficient (1.4→1.5) and static margin (6%→14%) for a design that proceeded to manufacturing 
(in progress).

- [📄 Report / CDR docs](./rc3-2026/)
- [💻 MATLAB sizing model](./rc3-2026/code)
